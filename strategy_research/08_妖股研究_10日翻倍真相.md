# 妖股研究: 10日翻倍票的真相

> 研究日期: 2026-07-22
> 核心结论: 妖股是从右下角启动涨到右上角的, 不是右上角继续涨成妖股

## 关键发现1: 短期暴涨的票大部分在右下角(未翻倍)

| 暴涨级别 | 右上角占比 | 右下角占比 |
|---|---|---|
| 3日涨≥20% | 32.4% | **67.6%** |
| 3日涨≥30% | 31.3% | **68.7%** |
| 5日涨≥30% | 39.6% | **60.4%** |
| 5日涨≥50% | 51.8% | 48.2% |
| 10日涨≥100%(翻倍) | 93.9% | 6.1% |

**只有10日翻倍以上的票才基本都在右上角。** 3-5天的短期暴涨,大部分是右下角票(从低位启动)。

## 关键发现2: 10日翻倍票, 翻倍前在右下角!

| 翻倍前(10日前)距翻倍线 | 数值 |
|---|---|
| 最小 | 39.3% |
| 25%分位 | 53.0% |
| **中位数** | **64.3%** |
| 75%分位 | 86.5% |

**翻倍前中位数距翻倍线只有64.3% -- 大部分翻倍票启动前价格只有翻倍线的一半多(右下角)。**

妖股生命周期: 右下角(64%位置) → 连续7个涨停 → 右上角(翻倍) → 继续10日涨100% → 2倍股

## 关键发现3: 翻倍票前10天几乎天天涨停

| 翻倍前10日涨停数 | 占比 |
|---|---|
| 5个涨停 | 6.1% |
| 6个涨停 | 19.7% |
| 7个涨停 | 21.8% |
| 8个涨停 | 21.1% |
| 9个涨停 | 14.3% |
| 10个涨停(天天板) | 14.3% |

**中位数7个涨停。** 妖股是连续涨停走出来的, 不是某天突然启动。

## 关键发现4: 识别妖股命中率只有0.2%

| 指标 | 数据 |
|---|---|
| 右上角+近5日有涨停的票(看着像妖股) | 10531只 |
| 10日后真翻倍 | **17只(0.2%)** |
| 10日后涨≥30% | 823只(7.8%) |
| 10日后跌≥10% | **3587只(34.1%)** |
| 10日后平均 | -0.0% |

**买"看起来像妖股"的右上角票, 期望收益是0 -- 34%概率跌10%+。**

## 给操作的意义

| 选择 | 目标 | 胜率 | 评价 |
|---|---|---|---|
| 抓3日30% | 右下角启动票 | ~46%(加过滤) | ✅ 可达, 从低位找 |
| 抓10日翻倍 | 连板妖股 | 0.2% | ❌ 几乎不可能识别 |
| 追右上角已翻倍 | 博继续涨 | 50%(均值0) | ❌ 34%跌10%+, 赌博 |

## 核心认知

1. **短期暴涨来自右下角(低位启动), 不是右上角(高位续涨)**
2. **妖股是从右下角涨到右上角的, 不是从右上角涨到2倍的**
3. **追右上角已翻倍的票, 期望收益是0** (一半赚一半亏)
4. **识别10日翻倍妖股命中率0.2%**, 这不是策略是买彩票
5. **想抓短期暴涨, 瞄准右下角(距翻倍线50-80%)+首板启动**
